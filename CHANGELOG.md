# Changelog

All notable changes to this project will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-05-16

### Added
- **Phase 4b: Creative Expert Roundtable.** Pool of twelve stylised expert lenses applied after framework critique. Five families: Creative Directors (Ogilvy, Bernbach, Wells Lawrence, Sagmeister, Trott), Design & Typography (Scher, Spiekermann), Persuasion & Behavior (Cialdini, Barden), Presentation & Storytelling (Duarte, Heath), UX & Clarity (Krug).
- Selection logic per asset type (pitch deck, landing page, LinkedIn post, CV, ad, brand identity, email).
- `references/expert-lens-principles.md`: documented core principles per lens with sources.
- Honesty rule: stylised lenses based on documented principles, never "Ogilvy says".
- "What this skill can NOT do" section in README for explicit limitations.
- LinkedIn cover asset for launch communications.

### Changed
- **Renamed from ad-pretest to Pretest Lab.** Repo, plugin, skill, all aligned.
- **Scope broadened from ads to design and communication assets.** Pitch decks, landing pages, LinkedIn posts, X threads, one-pagers, CVs, cover letters, brand identity work, email templates, plus the original ad use cases.
- **Anchor statement: "Communication only exists when it lands."** Tagline reset throughout README, plugin manifest and skill descriptions.
- **Positioning shifted to "pre-launch quality gate for design and communication assets".** Connected to the Creative Intelligence System operating model at gregoradammyszor.com.
- **Primary language is now English.** EN skill is the default at `skills/pretest-lab/`. German version at `skills/pretest-lab-de/`.
- **Persona terminology shifted to archetype-driven.** Phase 2 renamed from "Persona Panel" to "Archetype Panel".
- **Methods statement formalised as three frameworks:** Limbic Model (Häusel), Decoded neuromarketing (Barden), Creative Expert Roundtable (12 lenses).
- Framework dimensions now flex by asset type.
- Sample report rewritten in English.
